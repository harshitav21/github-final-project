## How to Contribute

Follow these steps to contribute:

1. **Fork** the repository.
2. **Clone** your fork to your local machine: git clone https://github.com/harshitav21/github-final-project.git
3. Navigate into the project folder: cd github-final-project
4. Create a new branch for your changes: git checkout -b feature-name
5. Make your changes and test them.
6. Commit your changes with a meaningful message: git commit -m "Add: short description of changes"
7. Push the branch to your GitHub fork: git push origin feature-name
8. Open a **Pull Request** from your branch to the main branch.
